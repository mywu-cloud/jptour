關於這份資料

目前地圖已接上全國46都道府県（不含沖繩）的例年見頃資料彙整，由 koyo.walkerplus.com 各都道府県頁面人工整理而成，僅擷取日期、狀態等事實性資料。由於2026年紅葉季尚未開始，目前狀態統一顯示為「青葉」；正式版將由 GitHub Actions 每日自動執行 Playwright 爬蟲，抓取即時見頃狀態並依日期分檔寫入，杜絕歷史覆寫問題。想看關西、關東各縣詳細名所清單，請至關西地區指南與關東地區指南。
